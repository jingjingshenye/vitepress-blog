## office 预览方案

|      文档格式      |         老的开源组件         |          替代开源组件           |
| :----------------: | :--------------------------: | :-----------------------------: |
|    word（docx）    |           mammoth            |        docx-preview(npm)        |
| powerpoint（pptx） |            pptxjs            |         pptxjs 改造开发         |
|   excel（xlsx）    |    sheetjs、handsontable     | exceljs(npm)、handsontable(npm) |
|     pdf（pdf）     |            pdfjs             |           pdfjs(npm)            |
|        图片        | jquery.verySimpleImageViewer |          v-viewer(npm)          |
