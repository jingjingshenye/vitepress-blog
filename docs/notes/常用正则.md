# 常用正则

## 1. 手机号码

```js
// 手机号码正则
const isPhone = /^1[3|4|5|7|8][0-9]{9}$/;
```

## 2. 邮箱

```js
// 邮箱正则
const isEmail =
  /^([a-zA-Z0-9_-])+@([a-zA-Z0-9_-])+((\.[a-zA-Z0-9_-]{2,3}){1,2})$/;
```

## 3. 身份证

```js
// 身份证正则
const isIdCard = /(^\d{15}$)|(^\d{18}$)|(^\d{17}(\d|X|x)$)/;
```

## 4. 车牌号

```js
// 车牌号正则
const isCarNo = /^[ 京津沪渝冀豫云辽黑湘皖鲁新苏浙赣鄂桂甘晋蒙陕吉闽贵粤青藏川宁琼使领 A-Z]{1}[A-Z]{1}(([0-9]{5}[DF]$)|([DF]([A-HJ-NP-Z0-9])[0-9
```
